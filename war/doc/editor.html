Editor Panel

At the top-middle of page the Editor is located. Editor section is the place that you write your code within and practice programming. Here is a sample code that you can copy to the editor by clicking on the >> button: 

public class HelloWorld{
 public static void main(String[] argsv) {
  System.out.println("Hello World");
 }
}

Look again at top-left of the screen. you will see “New Tab” button. You can add as many as independents tab you want and easily switch between them. Each tab can be used to write a separate program. You can change the “default” tab name to an arbitrary one by click on tab itself. This name will use as the file name when you try to save the code.
