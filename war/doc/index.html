Welcome

This tutorial is written in order to provide an quick accessible information to use key features of TML application in most efficient way possible. TML is an innovative web based solution targeting educational domain. It is developed for helping you to easily learn a new programming language. TML is an installing and configuration free platform on the cloud with no longer need to spend time consuming preparation and configuration of materials like IDE and compiler.

Getting start with TML By typing the TML URL into browser address-bar you will direct to application page ( you have already done this part). Here you have three different section naming tutorial panel, editor and console.
